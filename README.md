# Personal Portfolio v2

A professional, responsive personal portfolio website built to showcase engineering projects, technical research, and active development progress. This project focuses on high-performance static delivery and a clean, system-oriented user interface.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
* **Framework:** Bootstrap 5.3
* **Hosting:** Netlify
* **Typography:** Google Fonts (Silkscreen, Cutive Mono)

## Key Features
* **Dynamic Theming:** A custom-built Dark/Light mode toggle that persists across sessions using `localStorage`.
* **Systems Architecture Layout:** Modular design sections including:        
    * **Research Blueprint:** For academic and technical publications.
    * **Active Processes:** Dynamic progress tracking for ongoing learning (DSA, ML).
    * **Skill Matrix:** Categorized technical proficiency display.
* **Responsive Design:** Mobile-first approach using CSS `clamp()` and Bootstrap's grid system.
* **Case-Sensitive Optimization:** Pathing and asset management optimized for Linux-based hosting environments.
